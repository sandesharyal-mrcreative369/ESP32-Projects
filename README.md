\# 🚀 ESP32 Projects Collection



This repository contains a collection of ESP32 projects developed using \*\*MicroPython (Thonny IDE)\*\* and \*\*Arduino IDE\*\*. These projects are designed for beginners and enthusiasts to learn embedded systems and IoT development.



---



\## 📌 Project Categories



\### 🔹 MicroPython Projects (Thonny IDE)

\- 💡 LED Blink

\- 🔘 Push Button Interface

\- 📟 OLED Display (SSD1306)

\- 📏 Ultrasonic Sensor with OLED Display

\- 🔗 Combined Sensor-Based Projects



\### 🔹 Arduino IDE Projects

\- 💡 LED Blink

\- 🔘 Push Button

\- 📏 Ultrasonic Sensor

\- 📟 OLED Display Integration

\- 🤖 Multiple Sensor Projects
